\# Tâche 3 - Version Streamlit



Ce fichier décrit la Tâche 3 du projet "Diabetes Decision Tree" :



\- Ajout de l’interface Streamlit pour tester le modèle Decision Tree.

\- Scripts d’évaluation et visualisation (`plot\_tree.py`, `feature\_importance.py`).

\- Dataset utilisé : `dataset\_final.csv`.



\## Instructions



1\. Installer les dépendances : `pip install -r requirements.txt`

2\. Lancer l'application Streamlit : `streamlit run app/app\_streamlit.py`

3\. Tester le modèle avec l’interface interactive.
